# ElderCore Privacy Policy

_Last updated: 2025-11-03_

This Privacy Policy explains how ElderCore (the Discord bot) handles data when operating within Discord servers.

## 1. Data Sources
ElderCore receives data from Discord in accordance with server-granted permissions and the Discord API.

## 2. What We Process
- Message content only as needed for features (e.g., moderation triggers)
- User and server metadata (IDs, roles, channels) as provided by Discord
- Operational logs for reliability and abuse prevention

## 3. Storage and Retention
We strive to minimize storage and retain only what is necessary for functionality, security, and diagnostics.

## 4. Third Parties
If applicable, limited data may be shared with hosting or monitoring providers under appropriate safeguards.

## 5. Your Choices
Server administrators can remove the bot or adjust permissions at any time.

## 6. Changes
We may update this Policy. Material changes will be communicated via the repository or official channels.

## 7. Contact
Reach out via the contact methods listed in the repository or project website.
