# Building DApps on DRP

Decentralized Applications (DApps) are the future of web development. In this lesson, you'll learn how to build powerful DApps using DRP's infrastructure and tools.

## Learning Objectives

By the end of this lesson, you will be able to:
- Understand DApp architecture and components
- Set up a DRP development environment
- Build a simple DApp using DRP SDK
- Deploy and interact with smart contracts
- Implement user authentication and data management

## What are DApps?

**Decentralized Applications (DApps)** are applications that run on a decentralized network rather than a single computer. They combine:

- **Frontend**: User interface (React, Vue, Angular)
- **Smart Contracts**: Backend logic on blockchain
- **Blockchain Network**: Decentralized infrastructure
- **Web3 Integration**: Wallet connections and transactions

## DApp Architecture

### Traditional Web App
```
User → Frontend → Backend API → Database
```

### DApp Architecture
```
User → Frontend → Smart Contracts → Blockchain
     ↓
   Web3 Wallet
```

## Building Your First DApp

### 1. Project Setup
```bash
npx create-react-app my-dapp
cd my-dapp
npm install @drp/sdk ethers
```

### 2. DRP Client Configuration
```javascript
import { DRPClient } from '@drp/sdk';

const drpClient = new DRPClient({
  network: 'mainnet', // or 'testnet'
  rpcUrl: 'https://rpc.drp.network',
  chainId: 137
});
```

### 3. Wallet Connection
```javascript
import { ethers } from 'ethers';

const connectWallet = async () => {
  if (window.ethereum) {
    const provider = new ethers.providers.Web3Provider(window.ethereum);
    const signer = provider.getSigner();
    const address = await signer.getAddress();
    return { provider, signer, address };
  }
};
```

### 4. Smart Contract Interaction
```javascript
import { DRPContract } from '@drp/contracts';

const contract = new DRPContract({
  address: '0x...',
  abi: contractABI,
  signer: signer
});

// Call contract methods
const result = await contract.methods.getData();
```

## DApp Development Best Practices

### Security
- **Input Validation**: Validate all user inputs
- **Access Control**: Implement proper permissions
- **Error Handling**: Handle contract call failures
- **Gas Optimization**: Optimize for gas efficiency

### User Experience
- **Loading States**: Show progress indicators
- **Error Messages**: Provide clear error feedback
- **Transaction Status**: Track transaction progress
- **Mobile Responsive**: Ensure mobile compatibility

### Performance
- **Lazy Loading**: Load components on demand
- **Caching**: Cache contract data
- **Batch Operations**: Group multiple transactions
- **Optimistic Updates**: Update UI before confirmation

## DRP-Specific Features

### Digital Rights Management
```javascript
// Create digital rights
const rights = await drpClient.createRights({
  contentHash: '0x...',
  owner: userAddress,
  royaltyRate: 5 // 5%
});

// Transfer rights
await drpClient.transferRights(rightsId, newOwner);
```

### Activity Proofs
```javascript
// Generate activity proof
const proof = await drpClient.generateProof({
  activity: 'content_creation',
  timestamp: Date.now(),
  metadata: { type: 'image', size: '2MB' }
});

// Verify proof
const isValid = await drpClient.verifyProof(proof);
```

### Elder Quorum Integration
```javascript
// Submit governance proposal
const proposal = await drpClient.submitProposal({
  title: 'Update protocol parameters',
  description: 'Increase block size limit',
  type: 'parameter_change'
});

// Vote on proposal
await drpClient.vote(proposalId, true);
```

## Testing DApps

### Unit Testing
```javascript
import { DRPClient } from '@drp/sdk';

describe('DRP Client', () => {
  it('should connect to network', async () => {
    const client = new DRPClient({ network: 'testnet' });
    const isConnected = await client.isConnected();
    expect(isConnected).toBe(true);
  });
});
```

### Integration Testing
```javascript
describe('DApp Integration', () => {
  it('should create and transfer rights', async () => {
    const rights = await createRights();
    await transferRights(rights.id, newOwner);
    const owner = await getRightsOwner(rights.id);
    expect(owner).toBe(newOwner);
  });
});
```

## Deployment

### Smart Contract Deployment

npx hardhat compile

npx hardhat verify --network sepolia <contract-address>

```bash

npm run build

# Deploy to Vercel
vercel --prod

# Deploy to IPFS
npx ipfs-deploy dist
```
